## <Code size={28} style={{verticalAlign: 'text-bottom', marginRight: '8px', color: 'var(--brand-electric-blue)'}} /> Software SDK Overview

We provide SDKs for mainstream programming languages. All DAQ devices share the same API design logic. Click the links below for detailed tutorials:

| Languages | Features | Teaching Links |
| :------- | :------- | :----------- |
| **Python** | Open-source and easy-to-use API with consistent interfaces and rich examples. Ideal for all developers. | [Python Getting Started](https://github.com/WPC-Systems-Ltd/WPC_Python_driver_release) |
| **C#** | Open-source driver optimized for .NET. Uses intuitive open/read/write/close logic to significantly lower the learning curve. | [C# Getting Started](https://github.com/WPC-Systems-Ltd/WPC_CSharp_driver_release) |
| **LabVIEW** | Provides comprehensive VI packages and graphical panels to help engineers quickly integrate automated testing flows. | *Coming Soon* |

:::note
The SDK itself is **Open Source** and hosted on GitHub. It can be installed directly via package managers like Pypi and NuGet.
:::
