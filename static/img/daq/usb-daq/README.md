# USB-DAQ 圖片說明

## 圖片來源

本資料夾應包含以下圖片檔案：

### 總覽圖片
- `naming.png` - 產品命名規則圖
- `layout-ad-series.png` - AD/AOD/RD/CD 系列外觀圖
- `layout-d-series.png` - D/D-SNK 系列外觀圖
- `layout-td-series.png` - TD 系列外觀圖

### Pinout 圖片
- `usb-daq-ad-pinout.png` - USB-DAQ-AD 接腳圖
- `usb-daq-aod-pinout.png` - USB-DAQ-AOD 接腳圖
- `usb-daq-cd-pinout.png` - USB-DAQ-CD 接腳圖
- `usb-daq-d-pinout.png` - USB-DAQ-D 接腳圖
- `usb-daq-d-snk-pinout.png` - USB-DAQ-D-SNK 接腳圖
- `usb-daq-rd-pinout.png` - USB-DAQ-RD 接腳圖
- `usb-daq-td-pinout.png` - USB-DAQ-TD 接腳圖

### 接線說明圖
- `can-bus-wiring.png` - CAN Bus 接線圖
- `d-snk-do-wiring.png` - D-SNK DO 等效電路圖
- `d-snk-di-wiring.png` - D-SNK DI 等效電路圖
- `rtd-wiring.png` - RTD 感測器接線圖
- `thermocouple-connector.png` - 熱電偶連接器規格圖

## 取得圖片

這些圖片可從以下來源取得：

1. **Notion 說明書**：`Notion\WPC_USBDAQ_UserGuide.md` 中有圖片連結
2. **原始規格書**：向 WPC 索取原始規格書 PDF
3. **產品照片**：向 WPC 索取產品實物照片

## 注意事項

- 所有圖片建議使用 PNG 或 JPEG 格式
- 圖片寬度建議統一為 600-800 像素
- 確保圖片清晰可辨識
- Pinout 圖片需要包含完整的接腳標示
