# PYCO WIFI MINI AI

PYCO WIFI MINI AI is a mini Python controller focused on **8-channel Analog Input** and **9DoF IMU Attitude Estimation**, built-in **Wi-Fi**, suitable for lightweight data acquisition and sensing applications.

## Key Features

---

*   **Python Programmable**: Supports Stand-Alone Python application development.
*   **Analog Input (AI)**:
    - 8 channels simultaneous sampling, 16-bit single-ended.
    - Input range: ±10V.
*   **9DoF IMU**: Built-in attitude estimation (accelerometer/gyroscope/magnetometer).
*   **Web-based Configurator**: Web-based configuration interface.
*   **Wi-Fi**: 802.11 b/g/n (2.4 GHz to 2.5 GHz).
