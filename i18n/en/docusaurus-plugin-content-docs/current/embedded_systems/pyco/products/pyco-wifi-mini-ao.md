# PYCO WIFI MINI AO

PYCO WIFI MINI AO is a mini-sized Python controller focused on **8-channel analog output** and **9DoF IMU attitude estimation**, with built-in **Wi-Fi**, making it ideal for lightweight signal generation and control applications.

## Key Features

---

*   **Python Programmable**: Supports Stand-Alone Python application development.
*   **Analog Output (AO)**:
    - 8 channels simultaneous output, 16-bit single-ended.
    - Output range: ±10V.
*   **9DoF IMU**: Built-in attitude estimation (Accelerometer/Gyroscope/Magnetometer).
*   **Web-based Configurator**: Web-based configuration interface.
*   **Wi-Fi**: 802.11 b/g/n (2.4 GHz to 2.5 GHz).
