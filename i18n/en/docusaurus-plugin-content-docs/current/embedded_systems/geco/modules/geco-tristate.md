# 12ch Tri-state Switch Module

This module provides a 12-channel tri-state switching function.

## 1. Specifications

| Item                   | Specification                           |
| :--------------------- | :-------------------------------------- |
| **Relay Power**        | 12V DC                                 |
| **States**             | VCC / GND / NC (Tri-state)             |
