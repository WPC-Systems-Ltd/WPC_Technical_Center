# 12ch Tri-state Switch Module

This module provides 12-channel tri-state switch switching function.

## 1. Specifications

| Item | Specification |
| :--- | :--- |
| **Relay Power** | 12V DC |
| **State** | VCC / GND / NC (Tri-state) |
