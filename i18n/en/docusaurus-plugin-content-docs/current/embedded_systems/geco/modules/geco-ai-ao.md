# 24ch AI / 8ch AO Module

This module provides 24-channel analog input and 8-channel analog output functionality, suitable for multi-channel signal acquisition and control applications.

## 1. Specifications

### Analog Input (AI)

| Item                   | Specification                  |
| :--------------------- | :----------------------------- |
| **Connector**          | IDC 50P Connector              |
| **Channels**           | 24 ch (Simultaneous sampling) |
| **Resolution**         | 16 bit (Single-ended)          |
| **Input Range**        | -10 ~ 10 V                     |
| **Max Sampling Rate**  | 50 kHz                         |
| **Isolation**          | Channel to ground isolation    |

### Analog Output (AO)

| Item                   | Specification                  |
| :--------------------- | :----------------------------- |
| **Connector**          | IDC 50P Connector              |
| **Channels**           | 8 ch (Simultaneous output)    |
| **Resolution**         | 16 bit (Single-ended)          |
| **Output Range**       | -10 ~ 10 V                     |
