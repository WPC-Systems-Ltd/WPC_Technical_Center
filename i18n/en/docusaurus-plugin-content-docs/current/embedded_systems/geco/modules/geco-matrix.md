# 4x6 Matrix Switch Module

This module provides a 4x6 matrix switching function, utilizing mechanical relays.

## 1. Specifications

| Item                   | Specification                           |
| :--------------------- | :-------------------------------------- |
| **External Power**     | 24V DC                                 |
| **Matrix Size**        | Row: 4 & Column: 6                     |
| **Switch Type**        | Mechanical Relays                      |
