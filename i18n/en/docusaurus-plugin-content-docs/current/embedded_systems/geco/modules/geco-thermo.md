# 16ch Thermocouple Module

This module provides 16 channels of thermocouple input, supporting K-type thermocouples, suitable for temperature measurement applications.

## 1. Specifications

| Item                   | Specification                          |
| :--------------------- | :------------------------------------- |
| **Connector**          | IDC 50P Connector                      |
| **Channels**           | 16 ch                                  |
| **Sensor Type**        | K-type Thermocouple                    |
| **Max Sampling Rate**  | 10 Hz                                  |
| **Accuracy**           | ± 2°C                                  |
| **Measurement Range**  | -200 ~ 700 °C                          |
| **Resolution**         | 14 Bit, 0.25 °C                        |
| **Isolation**          | Channel to ground isolation            |
