# 24ch 2-wire Multiplexer Module

This module provides 24-channel 2-wire multiplexer switching function.

## 1. Specifications

| Item | Specification |
| :--- | :--- |
| **External Power** | 24V DC |
| **Channels** | 1 to 24ch (2-wire) |
| **Switch Type** | Mechanical Relays |
