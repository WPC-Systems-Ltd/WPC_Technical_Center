# 24ch 5V TTL Module

This module provides 24 channels of 5V TTL signals, suitable for high-speed logic signal control.

## 1. Specifications

| Item                   | Specification                           |
| :--------------------- | :-------------------------------------- |
| **Power**              | Internal power                         |
| **Channels**           | 24 ch                                  |
| **Signal Type**        | 5V TTL                                 |
| **Isolation**          | Opto-isolated (24V)                    |
| **Direction**          | Bi-directional IO                      |
