# 1ch Current Sense Amplifier

The WPC 1ch Current Sense Amplifier is a front-end module specifically designed to convert current signals into **voltage signals**. This ensures that measurement systems can obtain high-resolution data across various application scenarios.

### Key Features
*   **High-Precision Current Measurement**: Provides precision up to 20µA, ensuring accurate capture of microscopic current changes.
*   **High-Safety Isolation Protection**: Features a channel-to-ground isolation design and offers an input tolerance voltage of up to ±100V relative to analog ground, effectively protecting back-end equipment.
*   **Wide Measurement Range**: Supports a wide current input range of ±1A, converted into a standard ±10V output signal.
*   **Standardized Power Supply**: Uses industrial-standard 24VDC power supply, making it easy to integrate into existing control systems.

## Product Specifications and Features

The WPC 1ch Current Sense Amplifier provides compact and efficient current measurement functionality. Its main specifications are as follows:

*   **Power Input**: 24 VDC
*   **Input Range**: ±1 A
*   **Output Range**: ±10 V
*   **Measurement Precision**: 20 µA
*   **Isolation Design**: Channel-to-ground isolation
*   **Input Tolerance Voltage (relative to analog AGND)**: ±100 V
*   **Output Level**: Approx. 0.02 V
