# 1ch Strain Gauge Amplifier

The WPC Strain Gauge Amplifier (SGA) is a high-bandwidth signal amplification front-end module specifically designed for strain gauge measurements. It accurately amplifies the minute voltage variation signals generated by strain gauges into standard voltage outputs, localized for high-frequency strain and structural stress analysis through general analog input channels.

### Key Features
*   **High-Ratio Precision Amplification**: Provides a fixed 1000x gain (Gain), boosting microscopic strain signals into the measurement range.
*   **High-Bandwidth Measurement Capability**: Features a bandwidth of up to 300KHz, suitable for high-speed dynamic strain and shockwave observation.
*   **Dedicated Excitation Voltage**: Built-in stable 2V excitation voltage (Excitation voltage), optimizing the performance of 1/4 bridge sensors.
*   **High Electrical Isolation**: Power and signal feature internal isolation design, ensuring measurement accuracy and reducing ground loop interference.
*   **1/4 Bridge Support**: Specifically designed for 120 ohms 1/4 bridge strain gauge architectures.

## Product Specifications and Features

The WPC SGA delivers outstanding performance for high-frequency strain measurement applications. Its main specifications are as follows:

*   **Power Input**: 24 VDC
*   **Gain**: 1000
*   **Excitation Voltage**: 2 V
*   **Output Voltage Range**: ±2 V
*   **Bandwidth**: 300 KHz
*   **Base Noise**: Approx. 50 mV (at full bandwidth)
*   **Sensor Type**: 1/4 bridge, 120 ohms
*   **Isolation Design**: Internal isolation between power and signal (0V is not connected to AGND)

## Pinout and Hardware Interface

Below is the detailed pinout configuration for the WPC SGA:

| Pin Number | Label   | Function Description                       |
| :--------- | :------ | :----------------------------------------- |
| 1          | Sensor- | Connects to strain gauge (Load end -)      |
| 2          | Sensor+ | Connects to strain gauge (Load end +)      |
| 3          | AGND    | Signal Ground                              |
| 4          | Vout    | Signal Output                              |
| 5          | Vtune   | Offset adjustment reference (can be used for external compensation) |
| 6          | 24V     | Positive Power Input (24Vin)               |
| 7          | 0V      | Power Ground                               |
| VR         | Offset  | Offset adjustment resistor (Clockwise to increase offset) |
