# 3-Phase Voltage Isolator

The WPC 3-Phase Voltage Isolator is a signal processing front-end module specifically designed for high-voltage power system measurements. It safely isolates and attenuates high-voltage signals from single-phase or three-phase AC systems into low-voltage signals, facilitating connection with data acquisition systems (DAQ) or control equipment. It is suitable for motor drives and power grid monitoring applications.

### Key Features
*   **High-Voltage Measurement Capability**: Supports a high-voltage input range of up to ±1000V (Line-to-Neutral).
*   **High-Safety Isolation**: Features 1.5kV electrical isolation, effectively protecting back-end measurement equipment.
*   **Precise Attenuation Ratio**: Fixed 1/100 (L-to-N) attenuation ratio, scaling high-voltage signals down proportionally to ±10V.
*   **Wide Application Scenarios**: Suitable for single-phase/three-phase AC voltage measurements, AC motor parameter analysis, etc.
*   **Industrial Grade Standard**: Powered by 24VDC, compliant with standard industrial automation environments.

---

## Product Specifications and Features

The WPC 3-Phase Voltage Isolator delivers outstanding performance for high-voltage isolation measurement applications. Its main specifications are as follows:

*   **Power Supply**: 24 VDC
*   **Input Voltage Range**: ±1000 V (L-to-N)
*   **Output Voltage Range**: ±10 V
*   **Isolation**: 1.5 kV
*   **Attenuation Rate**: 1/100 (L-to-N)
*   **Application Range**: Single-phase/three-phase AC voltage measurement, AC motor testing
