## Main Screen Function Description

<div style={{textAlign: 'center'}}>
  <img src="/img/wpc_device_manager/main-screen.png" alt="WPC Device Manager Main Screen Introduction" width="60%" />
</div>

1.  **Device List**
    DAQ devices are divided into two categories according to the connection interface: **Network Devices** and **My Computer**.
    *   **Network Devices**: Lists Ethan series and WIFI series products.
    *   **My Computer**: Lists USB series products.

2.  **Test Panel**
    The system will dynamically display the corresponding test function buttons according to the selected device model.

3.  **Product Information**
    Displays detailed specifications of the device, including serial number, IP address, MAC address, and firmware version. Users can also configure parameters or view the pin definition diagram (Pinout) in this area.
    *   **⚠️ Icon**: Indicates that the current firmware is not the latest version, update is recommended.
    *   **✅ Icon**: Indicates that the current firmware is the latest version.

4.  **Specification and Status**
    Displays the hardware specifications and real-time connection status of the device.
    *   **Connected**: Device is connected.
    *   **Disconnected**: Device is not connected.
    *   **IP Conflict**: IP address conflict detected.
