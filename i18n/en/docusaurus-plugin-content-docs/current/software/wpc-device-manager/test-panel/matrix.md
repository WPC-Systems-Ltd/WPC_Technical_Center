# Matrix

Matrix panel provides users with control over matrix switch and multiplexer connection status. Through the visual interface, you can set the connection of intersection points between Rows and Columns.

<div style={{textAlign: 'center'}}>
  <img src="/img/wpc_device_manager/testPanel/matrix.png" alt="Matrix Test Panel" width="60%" />
</div>

## Operation Instructions

Click on the intersection points on the grid to toggle Connect or Disconnect status.

| Button | Function |
| :--- | :--- |
| **Start** | Enable Matrix control function and execute current connection settings. |
| **Stop** | Stop control. |
| **Disconnect All** | Immediately disconnect all intersection points. |
