## Main Screen Introduction

The main screen functions of SAPE are divided as follows:

<div style={{textAlign: 'center'}}>
  <img src="/img/wpc-stand-alone-editor/editor_page11_X116.png" alt="SAPE Main Screen Function Division" width="60%" />
</div>

1.  **Menu bar**:
    *   **a. File**: File management.
    *   **b. Connect**: Connection settings.
    *   **c. System**: System information.
2.  **Project File Path (Host PC)**: Displays the project file path on the host computer.
3.  **Project File (Host PC)**: Displays the list of project files on the host computer.
4.  **Project File (Target)**: Displays the list of files on the target device (WPC hardware).
5.  **Console**: Displays program output results and messages.
6.  **Editor**: Area for writing and editing Python code.
7.  **Action Buttons**: Contains function buttons such as run script, save script, etc.
