## Introduction

WPC Drone Driver is a driver package designed for LabVIEW, providing full API support for WPC Drone. Through this driver, users can easily control and monitor WPC Drone in the LabVIEW environment without needing to develop the underlying communication protocol from scratch.

## Download and Installation

### Step 1: Download WPC Drone Driver

Please go to the WPC official website to download the latest version of WPC Drone Driver.

> [Download WPC Drone Driver](https://www.wpc.com.tw/wpc-drone-driver.html)

  <img src="/WPC_Technical_Center/img/driver/wpc_drone_driver/wpc_drone_driver_download_page.png" alt="wpc_drone_driver_download_page" width="60%" />
</div>

### Step 2: Download VIPM Package Manager

Please go to the WPC official website to download the latest version of VIPM Package Manager.

> [Download VIPM Package Manager](https://www.wpc.com.tw/vipm-package-manager.html)

## Unzip and Install

### Step 1: Unzip the file
Unzip the downloaded `wpc_drone_driver.zip` file.

### Step 2: Install the driver through VIPM
Double-click the extracted `.vip` file to open the VIPM installation program.

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_device_driver/VIPM_install.png" alt="VIPM_install" width="60%" />
</div>

### Step 3: Execute Installation
Click **"Install"** or **"Upgrade"** to install the latest version of WPC Drone Driver.

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_drone_driver/wpc_drone_driver_install.png" alt="wpc_drone_driver_install" width="60%" />
</div>

### Step 4: Wait for the installation to complete
Please wait patiently during the installation and **do not close** the window until the installation is complete.
