# USB-DAQ-D

### Product Features
---

1. USB 2.0 Full-Speed Bus Powered
2. 26-channel 3.3V Digital I/O (5V Tolerant)
3. Supports I2C/SPI/UART/PWM/Counter
4. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](/ca1d9d570ec44097b4cdd0cebee0a77f)
