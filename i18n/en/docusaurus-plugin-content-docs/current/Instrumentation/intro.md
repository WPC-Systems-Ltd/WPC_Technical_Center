# gStack-RIO Product Introduction

## 1. Introduction

---

WPC's self-developed gStack product series adopts a Rack chassis design, specifically developed for distributed, high-density I/O applications. Each chassis has 8 slots, providing users with flexible options to combine various modules.

The module cards designed on the front panel can be quickly replaced, which is extremely convenient for system maintenance, replacement, and future functional expansion. Users using the gStack-RIO driver can develop LabVIEW applications through Host PC and Real-time applications to achieve efficient control and management.

## 2. gStack-RIO Chassis

---

The gStack-RIO chassis configuration is as follows:
*   Left side: Equipped with a gStack-RIO controller.
*   Right side: Equipped with 8 gStack-RIO module cards that can be freely combined as needed.
*   Power: 24VDC power is input through the European-style terminal block (green) at the bottom left. Most module cards do not require additional power supply, only a few power-type cards need external power supply support.

Control methods can be selected using Ethernet remote control (PC Remote), or through LabVIEW Real-time development of stand-alone applications. This design not only provides flexibility but also effectively supports various distributed, high-density I/O application requirements.
