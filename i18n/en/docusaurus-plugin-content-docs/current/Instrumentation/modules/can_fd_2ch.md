# cPCI 2ch-CAN FD

## 1. Introduction

---

The cPCI 2ch-CAN FD is a 2-channel CAN FD (Controller Area Network Flexible Data-Rate) card.

## 2. Features

---

*   Channels: 2 Channels
*   Connector: D-SUB 37P
*   Isolation: The two channels are isolated from each other.
*   Termination Resistor: Built-in 120 Ohm termination resistor.
