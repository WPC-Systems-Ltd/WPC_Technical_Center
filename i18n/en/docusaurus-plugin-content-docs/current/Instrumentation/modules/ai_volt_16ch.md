# cPCI 16ch-AI VOLT

## 1. Introduction

---

The cPCI 16ch-AI VOLT is a 32-channel single-ended (or 16-channel differential) +/- 10V voltage input analog I/O card.

## 2. Features

---

*   Channels: 32 Channels Single-ended or 16 Channels Differential
*   Input Range: +/- 10V
*   Resolution: 24 bit high resolution
*   Sampling Rate: Measurement speed up to 250Sps per channel
