# cPCI 8ch-RS232

## 1. Introduction

---

The cPCI 8ch-RS232 is a communication interface card with 8 channels of RS232.

## 2. Features

---

*   Channels: 8 Channels
*   Connector: D-SUB 37P
*   Isolation: Each channel is isolated from the others.
*   Baud Rate: Max Baud rate can reach around 500kbps.
