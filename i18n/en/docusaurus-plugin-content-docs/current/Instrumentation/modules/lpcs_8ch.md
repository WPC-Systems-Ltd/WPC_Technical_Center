# cPCI 8ch-LPCS

## 1. Introduction

---

The cPCI 8ch-LPCS is an 8-channel programmable current source module card.

## 2. Features

---

*   Channels: 8 Channels
*   Max Output: 0.45 Amps (0.45A)
*   Power Requirement: Requires external power supply from Vcs pin, maximum voltage up to 48VDC.
*   Monitoring Function: Can simultaneously monitor the output voltage and current of each channel, allowing users to perform compensation themselves to ensure output stability.
