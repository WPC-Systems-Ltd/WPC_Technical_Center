# cPCI 8ch-RS485 422

## 1. Introduction

---

The cPCI 8ch-RS485 422 is a communication card with 8 channels of RS485 or RS422.

## 2. Features

---

*   Channels: 8 Channels
*   Connector: D-SUB 37P
*   Isolation: Channels are isolated from each other.
*   Mode: RS-485 and RS-422 modes can be switched via software, each channel can be set to a different mode.
*   Baud Rate: Max Baud rate can reach around 500kbps.
