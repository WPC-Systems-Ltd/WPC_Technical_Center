# cPCI 32ch-DO

## 1. Introduction

---

The cPCI 32ch-DO is a 24V industrial isolated digital I/O card with 32 output channels.

## 2. Features

---

*   Channels: 32 Output Channels
*   Isolation: 24V Industrial Isolation
*   Connector: D-SUB 37P
*   Max Output Current: The maximum output current for each PIN is 350mA.
