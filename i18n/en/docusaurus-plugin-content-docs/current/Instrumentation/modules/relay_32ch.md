# cPCI 32ch-Relay

## 1. Introduction

---

The cPCI 32ch-Relay is a relay card with 32 channels (configurable as Form A or Form B).

## 2. Features

---

*   Channels: 32 Channels
*   Current Load: Each PIN can withstand up to 1A current.
*   Settings: Output contact form can be set to Normally Open (Form A) or Normally Closed (Form B) via Jumpers.
*   Characteristics: Low contact resistance characteristics make it very suitable for automated measurement applications.
