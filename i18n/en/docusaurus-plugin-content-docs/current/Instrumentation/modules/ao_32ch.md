# cPCI 32ch-AO

## 1. Introduction

---

The cPCI 32ch-AO is an analog I/O card with 32 channels of single-ended +/- 10V voltage output.

## 2. Features

---

*   Channels: 32 Channels (Single-ended)
*   Output Range: +/- 10V
*   Resolution: 16 bit
*   Update Rate: Measurement speed up to 250Sps per channel
