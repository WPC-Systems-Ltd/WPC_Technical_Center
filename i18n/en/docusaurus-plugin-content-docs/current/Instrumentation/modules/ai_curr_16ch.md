# cPCI 16ch-AI CURR

## 1. Introduction

---

This module is a 16-channel single-ended +/- 20mA current input I/O card.

## 2. Features

---

*   Channels: 16 Channels (Single-ended)
*   Input Range: +/- 20mA
*   Resolution: 24 bit high resolution
*   Sampling Rate: Measurement speed up to 250Sps per channel
