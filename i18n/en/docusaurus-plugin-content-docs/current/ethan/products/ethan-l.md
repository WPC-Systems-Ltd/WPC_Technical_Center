### Key Features

---

1. Power Input: 24VDC
2. 10/100 Base-T Ethernet Interface
3. 6-channel High Quality Mechanical Relay
4. 4-channel Form-A (NO) + 2-channel Form-C (NO+NC)
5. Max Current: 1A
6. Operate Time: 2ms (Bounce not included)
7. Release Time: 1ms (Bounce not included)
8. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](/ca1d9d570ec44097b4cdd0cebee0a77f)
