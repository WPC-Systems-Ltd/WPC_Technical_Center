### Key Features

---

1. Power Input: 24VDC
2. 10/100 Base-T Ethernet Interface
3. Analog Output (AO): 8-channel Synchronous Output & 16-bit Single-ended Resolution
4. Output Range: ±10V
5. Max Update Rate: 1kHz
6. Supports Waveform Generation
7. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](/ca1d9d570ec44097b4cdd0cebee0a77f)
