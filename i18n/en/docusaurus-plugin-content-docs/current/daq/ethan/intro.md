## Product Selection Guide

| Classification  | Model          | Key Features                                       |
| :-------------- | :------------- | :------------------------------------------------- |
| **Analog Input**| **Ethan-A**    | 8-channel 16-bit voltage input (±10V)             |
|                 | **Ethan-A2**   | 8-channel 24-bit voltage input (±13.75V)          |
|                 | **Ethan-I**    | 8-channel 24-bit current input (±20mA)            |
|                 | **Ethan-IA**   | 8-channel 24-bit voltage/current input            |
|                 | **Ethan-T**    | 7-channel thermocouple input (Supports K, J, N, R, S, T, E, B types) |
| **Analog Output**| **Ethan-O**    | 8-channel 16-bit voltage output (±10V), supports waveform generation |
| **Digital I/O** | **Ethan-D**    | 6-channel Relay (NPN drive), 8-channel DI, 2-channel PWM |
|                 | **Ethan-EX-D** | 16-channel DO, 16-channel DI, 2-channel PWM, 2-channel Counter |
|                 | **Ethan-L**    | 6-channel mechanical relay (4x Form-A, 2x Form-C) |
| **Others**      | **Ethan-P**    | 4-channel variable digital potentiometer (300Ω-50kΩ) |

> 💡 **Note**: For detailed specifications of some models, please refer to the [Product Specification Sheet](./specification.md).
