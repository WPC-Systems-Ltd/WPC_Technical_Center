Ethan-O provides **8-channel 16-bit analog output** (±10V), with an update rate of up to **1kHz**. It supports **arbitrary waveform generation**, making it suitable for signal simulation and control applications.

### Key Features

---

1. Power input: 24VDC
2. 10/100 Base-T Ethernet interface
3. Analog Output (AO): 8-channel synchronous output & 16-bit single-ended resolution
4. Output range: ±10V
5. Maximum update rate: 1kHz
6. Supports waveform generation
7. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://www.wpc.com.tw)
