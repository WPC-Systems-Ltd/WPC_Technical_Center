Ethan-A is an industrial-grade Ethernet data acquisition card, providing **8-channel synchronous 16-bit analog input**, with a maximum sampling rate of **20kHz**, suitable for high-speed signal measurement and monitoring applications.

---

### Key Features

---

1. Power input: 24VDC
2. 10/100 Base-T Ethernet interface
3. Analog Input (AI): 8-channel synchronous sampling & 16-bit single-ended resolution
4. Maximum sampling rate: 20kHz
5. Input range: ±10V
6. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://www.wpc.com.tw)
