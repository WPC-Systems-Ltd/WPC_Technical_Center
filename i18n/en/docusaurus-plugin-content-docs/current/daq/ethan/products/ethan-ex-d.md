Ethan-EX-D provides expanded digital I/O capabilities, including **16-channel DI**, **16-channel DO**, **2-channel PWM**, and **2-channel counter**, meeting complex industrial control requirements.

### Key Features

---

1. Power input: 24VDC
2. 10/100 Base-T Ethernet interface
3. Digital Output (DO): 16-channel NPN, 24V opto-isolated
4. Digital Input (DI): 16-channel NPN, 24V opto-isolated
5. 2-channel PWM output
6. 2-channel counter input
7. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://www.wpc.com.tw)
