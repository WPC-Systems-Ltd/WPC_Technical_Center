Ethan-A2 provides **8-channel 24-bit high-precision analog input**, with a sampling rate of up to **10kHz** and an input range of **±13.75V**, suitable for precision applications requiring high-resolution measurement.

### Key Features

---

1. Power input: 24VDC
2. 10/100 Base-T Ethernet interface
3. Analog Input (AI): 8-channel synchronous sampling & 24-bit single-ended resolution
4. Maximum sampling rate: 10kHz
5. Input range: ±13.75V
6. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://www.wpc.com.tw)
