Ethan-IA integrates **voltage and current dual-mode input**, providing **8-channel 24-bit resolution** with the flexibility to switch between measuring **±10V voltage** or **±20mA current** signals, offering multi-purpose utility in a single card.

### Key Features

---

1. Power input: 24VDC
2. 10/100 Base-T Ethernet interface
3. Current Input: 8-channel multiplexed switching & 24-bit resolution
4. Voltage Input: 8-channel multiplexed switching & 24-bit resolution
5. Maximum update rate: 600Hz (Single channel)
6. Current input range: ±20mA
7. Voltage input range: ±10V
8. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://www.wpc.com.tw)
