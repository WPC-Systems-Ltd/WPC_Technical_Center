# 4x6 Matrix Switch Module

This module provides 4x6 matrix switch function, allowing free switching of Row and Column connection status.


## Specifications

| Item | Specification |
| :--- | :--- |
| **External Power** | 24 VDC |
| **Connector** | IDC 50P Connector |
| **Matrix Size** | Row 4 x Column 6 |
| **Switch Element** | Mechanical Relay |
