# 4x6 Matrix Switch Module

This module provides 4x6 matrix switching functionality, allowing free switching of connection states between Rows and Columns.


## Specifications

| Item              | Specification      |
| :---------------- | :----------------- |
| **External Power**| 24 VDC             |
| **Connector**     | IDC 50P Connector  |
| **Matrix Size**   | Row 4 x Column 6   |
| **Switching Element**| Mechanical Relay |
