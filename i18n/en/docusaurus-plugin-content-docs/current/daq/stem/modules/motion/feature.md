# 4-axis Motion Control Module

This module provides 4-axis motion control function, supporting Pulse Command control.

## Features

*   **Control Mode**: Pulse/Direction or CW/CCW
*   **Output Frequency**: Differential output up to 8 MPPS
*   **Encoder Input**: Supports ABZ phase (Max 12 MHz)
*   **Limit and Home**: Supports Home / FWD / REV limit input
*   **Servo Control**: Supports Servo On / Alarm Clear output
*   **Advanced Features**:
    *   Breakpoint output and Position Capture
    *   Trapezoidal and S-curve velocity profile
    *   Linear/circular/helical interpolation
    *   Multi-axes sync move
