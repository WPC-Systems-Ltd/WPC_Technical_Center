# Specifications

## Pulse Command Output

| Item | Specification |
| :--- | :--- |
| **Axes** | 4 Axes |
| **Control Type** | Open loop or P-command |
| **Control Mode** | Absolute, Relative, Velocity modes |
| **Signal Type** | RS-422 or Line-driver output |
| **Output Mode** | Step/Direction or CW/CCW |
| **Max Frequency** | 8 MHz |
| **Voltage Range** | 0 ~ 5V Optocoupler Isolation |
| **Active State** | Programmable (Active High / Active Low) |
| **Trajectory Function** | S-curve, Position, Blending, Velocity Blending |
| **Contour Function** | 2D, 3D Contouring |
| **Interpolation Function** | Linear, Circular, Helical interpolation |
