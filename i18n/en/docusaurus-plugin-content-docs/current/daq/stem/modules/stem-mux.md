# 24ch 2-wire Multiplexer Module

This module provides 24-channel multiplexing function, suitable for multi-point signal switching measurement.

## Features


| Item | Specification |
| :--- | :--- |
| **External Power** | 24 VDC |
| **Connector** | IDC 50P Connector |
| **Channels** | 24 ch (2-wire) |
| **Switch Element** | Mechanical Relay |
