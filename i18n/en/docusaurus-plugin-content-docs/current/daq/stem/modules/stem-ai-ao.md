# 24ch AI / 8ch AO Module

This module provides 24-channel Analog Input and 8-channel Analog Output functions.

## Specifications

### Analog Input (AI)

| Item | Specification |
| :--- | :--- |
| **Connector** | IDC 50P Connector |
| **Channels** | 24 ch (Simultaneous Sampling) |
| **Resolution** | 16 bit (Single-ended) |
| **Input Range** | -10 ~ 10 V |
| **Max Sampling Rate** | 12 kHz |

### Analog Output (AO)

| Item | Specification |
| :--- | :--- |
| **Connector** | IDC 50P Connector |
| **Channels** | 8 ch (Simultaneous Output) |
| **Resolution** | 16 bit (Single-ended) |
| **Output Range** | 0 ~ 5 V |
