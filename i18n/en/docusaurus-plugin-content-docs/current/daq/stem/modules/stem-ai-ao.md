# 24ch AI / 8ch AO Module

This module provides 24-channel analog input and 8-channel analog output functions.

## Specifications

### Analog Input (AI)

| Item              | Specification                  |
| :---------------- | :----------------------------- |
| **Connector**     | IDC 50P Connector              |
| **Channels**      | 24 ch (Synchronous sampling)  |
| **Resolution**    | 16 bit (Single-ended input)    |
| **Input Range**   | -10 ~ 10 V                     |
| **Max Sampling Rate**| 12 kHz                         |

### Analog Output (AO)

| Item              | Specification                  |
| :---------------- | :----------------------------- |
| **Connector**     | IDC 50P Connector              |
| **Channels**      | 8 ch (Synchronous output)      |
| **Resolution**    | 16 bit (Single-ended output)   |
| **Output Range**  | 0 ~ 5 V                        |
