# STEM Module Series

STEM is a remote controller that supports a 10/100 Ethernet interface and is equipped with 4 slots. Through STEM's 4 slots (C1 ~ C4), users can select different types of modules for functional expansion.

Common module functions include the following:
*   4-axis motion control module
*   24ch-AI & 8ch-AO module
*   24ch DO module
*   24ch DI module
*   12ch DI / 12ch DO module
*   4x6 Matrix Switch module
*   24ch-2wire Multiplexer module

## System Specifications

| Item              | Specification                           |
| :---------------- | :-------------------------------------- |
| **Power Input**   | 24V DC input                            |
| **Network Interface**| 10/100 Base-T Ethernet (RJ45)           |
| **Expansion Slots**| 4 slots (Slot C1 ~ C4)                  |
| **Slot Interface** | IDC 50P, 3.3V DIO                       |
| **Supported Software**| LabVIEW, Python, C#, WPC Device Manager |
| **Dimensions**    | (Please refer to each module's appearance drawing) |
