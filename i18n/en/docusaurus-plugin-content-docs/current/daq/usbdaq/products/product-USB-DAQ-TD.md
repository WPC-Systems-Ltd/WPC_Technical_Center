# USB-DAQ-TD

### Product Features
---

1. USB 2.0 Full-Speed Bus Powered
2. 21-channel 3.3V Digital I/O (5V Tolerant)
3. Supports I2C/SPI/UART/PWM/Counter
4. 2-channel Thermocouple Input
5. Supported Sensor Types: K, J, N, R, S, T, E, & B
6. Supports [Python](https://wpc-systems-ltd.github.io/WPC_Python_driver_release/), [C#](https://wpc-systems-ltd.github.io/WPC_CSharp_driver_release/), and [LabVIEW](https://wpc-systems-ltd.github.io/WPC_LabVIEW_driver_release/)

### Thermocouple Wiring Guide
---

The USB-DAQ-TD uses standard Thermocouple connectors; simply plug in the connector to use. If you purchase thermocouple wire separately and connect it to a standard connector yourself, you might accidentally reverse the polarity. If the positive and negative poles are reversed, the DAQ temperature reading will behave oppositely: showing a temperature drop when heating, and a rise when cooling. In this case, simply swap the positive and negative connections.
