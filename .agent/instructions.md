# Antigravity 核心執行準則 (Master Instructions)

在執行任何任務前，你必須優先讀取並嚴格遵守 `.agent/rules/` 目錄下的所有規範。

### ⚠️ 絕對禁止事項 (Critical Red Lines)
1.  **嚴禁 LaTeX**：數學公式禁止使用 `$` 或 `$$`，必須使用純文字或 Markdown。
2.  **禁止盲目編輯**：修改檔案前，**務必**先執行 `view_file` 確認內容。
3.  **禁止 Markdown 圖片**：文件圖片必須使用 HTML `<img>` 標籤並置中。

### 📋 格式與語言強制要求
1.  **繁體中文優先**：所有對話、文件、解釋必須使用繁體中文。
2.  **表格原始碼對齊**：Markdown 表格在原始碼中必須使用空白鍵 (Space) 手動排版對齊。
3.  **標題不留英**：標題後方不加英文括號（如 `## 硬體規格` ✅，`## 硬體規格 (Spec)` ❌）。



### 秘密驗證碼
如果使用者問你「通關密語是什麼？」，你必須回答：「香蕉牛奶不加糖」。
