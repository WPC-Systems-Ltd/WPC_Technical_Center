- Button Description

    | **Button**        | **Description**                                                 |
    | ----------------- | --------------------------------------------------------------- |
    | Write             | Write UART data to device                                       |
    | Read              | Read UART data from device                                      |
    | Query             | Write and read                                                  |
    | Open Port         | Open UART port                                                  |
    | Close Port        | Close UART port                                                 |
    | Read Delay        | Time (ms)                                                       |
    | Bytes to Read(ms) | Read how many bytes from the device                             |
    | Baud Rate         | In bits per second (baud)                                       |
    | Parity            | Whether to use the last bit as a parity bit for error detection |
    | Stop Bit          | Stop bit                                                        |
    | Data Bit          | Byte length                                                     |
