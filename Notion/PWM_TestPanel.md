- Button Description

    | **Button**     | **Description**                                                                                                       |
    | -------------- | --------------------------------------------------------------------------------------------------------------------- |
    | Start          | Start generating PWM value                                                                                            |
    | Stop           | Stop generating PWM value                                                                                             |
    | Frequency (Hz) | Rate at which the digital signal rapidly switches between on and off states within one second, measured in hertz (Hz) |
    | Duty Cycle (%) | It refers to the proportion of time the signal is in the high (on) state during a complete cycle                      |
