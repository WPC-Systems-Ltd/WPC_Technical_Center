## 簡介

WPC Drone Driver 是一套專為 LabVIEW 設計的驅動程式套件，提供 WPC Drone 的完整 API 支援。透過此驅動程式，使用者可以在 LabVIEW 環境中輕鬆控制與監控 WPC Drone，無需從頭開發底層通訊協定。

## 下載與安裝

### 步驟 1: 下載 WPC Drone Driver

請前往 WPC 官方網站下載最新版本的 WPC Drone Driver。

> [下載 WPC Drone Driver](https://www.wpc.com.tw/wpc-drone-driver.html)

  <img src="/WPC_Technical_Center/img/driver/wpc_drone_driver/wpc_drone_driver_download_page.png" alt="wpc_drone_driver_download_page" width="60%" />
</div>

### 步驟 2: 下載 VIPM Package Manager

請前往 WPC 官方網站下載最新版本的 VIPM Package Manager。

> [下載 VIPM Package Manager](https://www.wpc.com.tw/vipm-package-manager.html)

## 解壓縮與安裝

### 步驟 1: 解壓縮檔案
請解壓縮下載的 `wpc_drone_driver.zip` 檔案。

### 步驟 2: 透過 VIPM 安裝驅動程式
請連按兩下解壓縮後的 `.vip` 檔案，以開啟 VIPM 安裝程式。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_device_driver/VIPM_install.png" alt="VIPM_install" width="60%" />
</div>

### 步驟 3: 執行安裝
請點擊 **"Install"** 或 **"Upgrade"** 以安裝最新版本的 WPC Drone Driver。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_drone_driver/wpc_drone_driver_install.png" alt="wpc_drone_driver_install" width="60%" />
</div>

### 步驟 4: 等待安裝完成
安裝過程中請耐心等待，且**不要關閉**視窗，直到安裝完成。
