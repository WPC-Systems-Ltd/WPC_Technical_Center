## 開啟 LabVIEW 與驅動程式面板

開啟 LabVIEW，在程式區 (Block Diagram) 點擊滑鼠右鍵，展開 Functions 面板。您將會看到 **"WPC Drivers"** 分類，GECO 的驅動程式皆位於此處。
