## 簡介

GECO Driver 是一套專為 LabVIEW 設計的驅動程式套件，提供 GECO 各項產品的完整 API 支援。透過此驅動程式，使用者可以在 LabVIEW 環境中輕鬆控制與監控 GECO 裝置。


## 下載與安裝

### 步驟 1: 下載 GECO Driver

請前往 WPC 官方網站下載最新版本的 GECO Driver。

> [下載 GECO Driver](https://www.wpc.com.tw/geco-driver.html)

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/geco_driver/geco_driver_download_page.png" alt="geco_driver_download_page" width="60%" />
</div>

:::note
目前 GECO Driver 支援 LabVIEW 2015-2019 版本。若需要支援更高版本的 LabVIEW，請聯絡 WPC 技術團隊。
:::

### 步驟 2: 下載 VIPM Package Manager

請前往 WPC 官方網站下載最新版本的 VIPM Package Manager。

> [下載 VIPM Package Manager](https://www.wpc.com.tw/vipm-package-manager.html)

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/geco_driver/VIPM_download_page.png" alt="VIPM_download_page" width="60%" />
</div>

## 解壓縮與安裝

### 步驟 1: 解壓縮檔案
請解壓縮下載的 `geco_driver.zip` 檔案。

### 步驟 2: 透過 VIPM 安裝驅動程式
請連按兩下解壓縮後的 `.vip` 檔案，以開啟 VIPM 安裝程式。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_device_driver/VIPM_install.png" alt="VIPM_install" width="60%" />
</div>

### 步驟 3: 執行安裝
請點擊 **"Install"** 或 **"Upgrade"** 以安裝最新版本的 WPC Device Driver。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_device_driver/VIPM_install1.png" alt="VIPM_install1" width="60%" />
</div>

### 步驟 4: 等待安裝完成
安裝過程中請耐心等待，且**不要關閉**視窗，直到安裝完成。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/driver/wpc_device_driver/VIPM_install2.png" alt="VIPM_install2" width="60%" />
</div>
