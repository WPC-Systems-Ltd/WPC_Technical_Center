## LED 狀態說明

透過機身上的 LED 燈號，您可以快速判斷裝置當前狀態：

| LED 顏色 | 狀態說明                                                |
| :------- | :------------------------------------------------------|
| **紅色** | 持續亮紅燈表示 AP 模式，此時需要進入 Web 介面設定  Wi-Fi。 |
| **綠色** | 持續亮綠燈表示裝置處於可連線狀態 (待機模式)。              |
| **藍色** | 持續亮藍燈表示裝置已成功連線。                            |
