# PYCO WIFI 系列螢幕顯示 `No wifi connection`

### 功能完整，無需擔心！

PYCO WIFI 系列設計為**高度獨立運作**的設備。即使螢幕顯示 `No wifi connection`，這僅表示**開發板目前未連接到無線網路**。這**不影響**開發板的核心功能：

1.  **獨立運作:**
    - 即使沒有 Wi-Fi 連接，產品的**所有主要功能**（如：數據採集、I/O 控制、執行內建程式）仍將**正常運作**。
    - 開發板將繼續獨立執行您的自定義程式碼或預設功能。

2.  **何時需要 Wi-Fi？**
    - 僅當您需要使用**雲端數據上傳**、**遠端控制**或**透過網頁介面進行設定**等功能時，才需要 Wi-Fi 連線。

3.  **如何清除訊息：**
    - 如果您希望移除此訊息，您可以透過程式碼將開發板連接到 Wi-Fi 網路，或者選擇**關閉螢幕顯示**或在您的應用程式中**修改顯示內容**。

    :::

:::note
即使沒有 Wi-Fi，您的 PYCO 開發板仍是一個功能齊全的數據採集與控制單元。請放心使用！
:::
