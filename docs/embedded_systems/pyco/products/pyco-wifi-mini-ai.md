# PYCO WIFI MINI AI

PYCO WIFI MINI AI 是一款迷你型 Python 控制器，專注於 **8 通道類比輸入** 與 **9DoF IMU 姿態估計**，內建 **Wi-Fi**，適合輕量級數據採集與感測應用。

## 主要特色

---

*   **Python Programmable**: 支援 Stand-Alone Python 應用程式開發。
*   **類比輸入 (AI)**:
    - 8 通道同步取樣，16-bit 單端。
    - 輸入範圍: ±10V。
*   **9DoF IMU**: 內建姿態估計 (加速度計/陀螺儀/磁力計)。
*   **Web-based Configurator**: 網頁式設定介面。
*   **Wi-Fi**: 802.11 b/g/n (2.4 GHz to 2.5 GHz)。
