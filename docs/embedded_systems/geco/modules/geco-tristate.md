# 12ch Tri-state Switch 模組

本模組提供 12 通道三態開關切換功能。

## 1. 規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **繼電器電源** | 12V DC                                 |
| **狀態**       | VCC / GND / NC (Tri-state)             |
