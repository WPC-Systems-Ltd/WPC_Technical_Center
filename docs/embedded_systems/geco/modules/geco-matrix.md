# 4x6 Matrix Switch 模組

本模組提供 4x6 矩陣開關功能，採用機械式繼電器。

## 1. 規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **外部電源**   | 24V DC                                 |
| **矩陣規模**   | Row: 4 & Column: 6                     |
| **開關類型**   | Mechanical Relays                      |
