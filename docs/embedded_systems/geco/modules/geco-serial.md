# RS-232 / 422 / 485 模組

本模組提供多通道序列通訊介面，支援 RS-232、RS-422 與 RS-485 標準，並具備通道間隔離功能。

## 1. 規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **電源**       | No external power required             |
| **RS-232**     | 4 ch                                   |
| **RS-485/422** | 4 ch (Configurable)                    |
| **隔離**       | Channel to channel isolation           |
