# 24ch 2-wire Multiplexer 模組

本模組提供 24 通道 2-wire 多工切換功能。

## 1. 規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **外部電源**   | 24V DC                                 |
| **通道數**     | 1 to 24ch (2-wire)                     |
| **開關類型**   | Mechanical Relays                      |
