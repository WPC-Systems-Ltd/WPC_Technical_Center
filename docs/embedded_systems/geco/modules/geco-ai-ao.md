# 24ch AI / 8ch AO 模組

本模組提供 24 通道類比輸入與 8 通道類比輸出功能，適用於多通道訊號擷取與控制應用。

## 1. 規格

### 類比輸入 (AI)

| 項目           | 規格                           |
| :------------- | :----------------------------- |
| **連接器**     | IDC 50P Connector              |
| **通道數**     | 24 ch (同步取樣)               |
| **解析度**     | 16 bit (單端輸入 Single-ended) |
| **輸入範圍**   | -10 ~ 10 V                     |
| **最大取樣率** | 50 kHz                         |
| **隔離**       | Channel to ground isolation    |

### 類比輸出 (AO)

| 項目           | 規格                           |
| :------------- | :----------------------------- |
| **連接器**     | IDC 50P Connector              |
| **通道數**     | 8 ch (同步輸出)                |
| **解析度**     | 16 bit (單端輸出 Single-ended) |
| **輸出範圍**   | -10 ~ 10 V                     |
