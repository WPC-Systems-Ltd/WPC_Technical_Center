# 16ch Thermocouple 模組

本模組提供 16 通道熱電偶輸入，支援 K-type 熱電偶，適用於溫度量測應用。

## 1. 規格

| 項目           | 規格                                   |
| :------------- | :------------------------------------- |
| **連接器**     | IDC 50P Connector                      |
| **通道數**     | 16 ch                                  |
| **感測器類型** | K-type Thermocouple                    |
| **最大取樣率** | 10 Hz                                  |
| **精準度**     | ± 2°C                                  |
| **量測範圍**   | -200 ~ 700 °C                          |
| **解析度**     | 14 Bit, 0.25 °C                        |
| **隔離**       | Channel to ground isolation            |
