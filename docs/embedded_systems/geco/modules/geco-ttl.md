# 24ch 5V TTL 模組

本模組提供 24 通道 5V TTL 訊號，適用於高速邏輯訊號控制。

## 1. 規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **電源**       | Internal power                         |
| **通道數**     | 24 ch                                  |
| **訊號類型**   | 5V TTL                                 |
| **隔離**       | Opto-isolated (24V)                    |
| **方向**       | Bi-directional IO                      |
