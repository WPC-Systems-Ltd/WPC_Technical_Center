# 3-Phase Voltage Isolator (三相電壓隔離器)

WPC 三相電壓隔離器是一款專為高壓電力系統測量設計的信號處理前端模組。它能將單相或三相交流系統中的高電壓訊號安全地隔離並衰減為低電壓訊號，以便於與資料擷取系統 (DAQ) 或控制設備連接，適用於馬達驅動與電網監測應用。

### 核心特色
*   **高壓量測能力**：支援最高 ±1000V (Line-to-Neutral) 的高壓輸入範圍。
*   **高安全性隔離**：具備 1.5kV 的電氣隔離度，有效保護後端量測設備。
*   **精確衰減比例**：固定 1/100 (L-to-N) 的衰減率，將高壓訊號等比例縮小至 ±10V。
*   **寬廣應用場景**：適用於單相/三相 AC 電壓量測、交流馬達參數分析等。
*   **工業級標準**：採用 24VDC 供電，符合標準工業自動化環境。

---

## 產品規格與特點

WPC 三相電壓隔離器針對高壓隔離量測提供優異的性能表現，其主要規格如下：

*   **供電電源**：24 VDC
*   **輸入電壓範圍**：±1000 V (L-to-N)
*   **輸出電壓範圍**：±10 V
*   **隔離度 (Isolation)**：1.5 kV
*   **衰減率 (Attenuation Rate)**：1/100 (L-to-N)
*   **應用範圍**：單相/三相交流電壓量測、交流馬達測試
