## 主畫面介紹

SAPE 的主畫面功能分區如下：

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/wpc-stand-alone-editor/editor_page11_X116.png" alt="SAPE 主畫面功能分區" width="60%" />
</div>

1.  **Menu bar (選單列)**:
    *   **a. File**: 檔案管理。
    *   **b. Connect**: 連線設定。
    *   **c. System**: 系統資訊。
2.  **Project File Path (Host PC)**: 顯示主機電腦上的專案檔案路徑。
3.  **Project File (Host PC)**: 顯示主機電腦上的專案檔案列表。
4.  **Project File (Target)**: 顯示目標裝置 (WPC 硬體) 上的檔案列表。
5.  **Console (控制台)**: 顯示程式輸出的結果與訊息。
6.  **Editor (編輯器)**: 撰寫與編輯 Python 程式碼的區域。
7.  **Action Buttons (操作按鈕)**: 包含執行腳本、儲存腳本等功能按鈕。

請使用 USB 線直接將 WPC 硬體連接至主機電腦 (Host PC)。
