# DPOT

DPOT (Digital Potentiometer) 面板提供使用者控制數位可變電阻，可應用於類比電路參數調整或感測器模擬。

## 支援產品

| 產品名稱                                            | 說明                                         |
| :-------------------------------------------------- | :------------------------------------------- |
| [`Ethan-P`](../../../daq/ethan/products/ethan-p.md) | 4 通道數位可變電阻輸出 (範圍：`300 ~ 50k Ω`) |

## 操作說明

您可以在介面中透過輸入框直接設定電阻值，或使用滑桿進行調整。

### 控制按鈕

| 按鈕           | 功能                           |
| :------------- | :----------------------------- |
| **Start**      | 啟用 DPOT 控制功能。           |
| **Stop**       | 停止 DPOT 控制功能。           |
| **Update**     | 將設定的電阻值寫入至指定通道。 |
| **Set all to** | 將所有通道設定為統一數值。     |
