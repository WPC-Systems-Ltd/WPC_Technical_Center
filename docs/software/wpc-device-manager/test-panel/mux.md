# MUX

MUX (Multiplexer) 面板提供使用者控制多工器的通道切換。

<div style={{textAlign: 'center'}}>
  <img src="/img/wpc_device_manager/testPanel/mux.png" alt="MUX Test Panel" width="60%" />
</div>

## 操作說明

| 按鈕             | 功能                 |
| :--------------- | :------------------- |
| **Start**        | 啟用 MUX 控制功能。   |
| **Stop**         | 停止控制。           |
| **Disconnect All** | 斷開所有通道的連接。 |
