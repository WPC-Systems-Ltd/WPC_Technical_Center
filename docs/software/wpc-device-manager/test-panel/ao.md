# AO

AO (Analog Output) 面板提供使用者進行電壓或電流的類比訊號輸出控制。透過此介面，您可以手動設定各通道的輸出數值，或同步更新多個通道。

## 支援產品

本功能僅支援具備類比輸出 (AO) 功能的裝置，請確認您的裝置型號及支援的輸出範圍：

| 產品名稱                                                                | 輸出規格說明                    |
| :---------------------------------------------------------------------- | :------------------------------ |
| [`Ethan-O`](../../../daq/ethan/products/ethan-o.md)                     | 8 通道電壓輸出 (範圍：`+/-10V`) |
| [`USB-DAQ-F1-AOD`](../../../daq/usbdaq/products/product-USB-DAQ-AOD.md) | 8 通道電壓輸出 (範圍：`+/-5V`)  |

## 操作說明

### 1. 單通道數值設定

在介面中直接調整各通道 (Channel) 的數值輸入框或滑桿，即可設定該通道的目標輸出電壓。

- **Update**：輸入數值後，按下該通道旁的 `Update` 按鈕，裝置將立即輸出設定的電壓值。

### 2. 多通道同步設定 (Set all to)

若需將所有通道設定為相同數值，請使用上方的 **Set all to** 功能：
1. 在 `Set all to` 輸入框中填入目標電壓值。
2. 按下 `Set all to` 按鈕，所有通道的數值將同時更新。

### 3. 開始與停止輸出

透過下方的控制按鈕來啟動或停止訊號輸出功能：

| 按鈕       | 功能                                                                               |
| :--------- | :--------------------------------------------------------------------------------- |
| **Start**  | 啟動 AO 功能。裝置將開始依照設定值輸出訊號。                                       |
| **Stop**   | 停止 AO 功能。停止後，裝置將維持最後一次輸出的電壓值或歸零（視裝置硬體設定而定）。 |
