## 問題描述

在 WDM 中可以搜尋到網路裝置，且能成功 Ping 通裝置，但無法建立 TCP 連線，狀態列顯示為 **"Occupied"**。

## 可能原因

此問題通常源於 **IP 位址衝突** 或 **路由混淆**：
1.  電腦端的 Wi-Fi 網卡與有線網卡 (Ethernet) 設定了相同的網段。
2.  電腦端 IP 與裝置 IP 完全相同。

## 解決方案

請檢查電腦的網路介面卡設定，確保：
1.  **網段區隔**：Wi-Fi 網卡與有線網卡應使用不同的網段（例如 Wi-Fi 使用 `192.168.1.x`，則有線網卡應避開此網段）。
2.  **IP 唯一性**：確保電腦端的 IP 位址（例如 `192.168.1.20`）與 WPC 裝置的 IP 位址（例如 `192.168.1.110`）不重複。

:::tip
請檢查電腦上的其他虛擬網卡或網路介面，確認是否有其他網卡被設定為相同的網段。
:::
