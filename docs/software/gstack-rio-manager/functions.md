# 通用功能

## 重新整理

點擊 **"Refresh"** 按鈕可重新讀取並更新 gStack-RIO 裝置資訊。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/wpc-gstack-rio-manager/refresh_button.png" alt="重新整理" width="60%" />
</div>


## 腳位圖

點擊 **"Pinout"** 按鈕以顯示腳位圖。視窗大小可根據使用者需求調整。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/wpc-gstack-rio-manager/pinout_button.png" alt="腳位圖視窗" width="60%" />
</div>

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/wpc-gstack-rio-manager/pinout_button_popout.png" alt="腳位圖視窗" width="60%" />
</div>

## 重新啟動

若需軟體重啟裝置，請點擊 **"Reboot"** 按鈕。

<div style={{textAlign: 'center'}}>
  <img src="/WPC_Technical_Center/img/wpc-gstack-rio-manager/reboot_button.png" alt="重新啟動" width="60%" />
</div>

將模組設定在特定的插槽中。
