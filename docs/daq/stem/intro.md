# STEM Series

STEM 是一款支援 10/100 乙太網路介面，具備 4 個插槽的遠端控制器。透過 STEM 的 4 個插槽 (C1 ~ C4)，使用者可以選擇不同類型的模組，進行功能的擴充。

常見的模組功能包含以下：
*   4 軸運動控制模組
*   24ch-AI & 8ch-AO 模組
*   24ch DO 模組
*   24ch DI 模組
*   12ch DI / 12ch DO 模組
*   4x6 Matrix Switch 模組
*   24ch-2wire Multiplexer 模組

## 系統規格

| 項目           | 規格                                    |
| :------------- | :-------------------------------------- |
| **電源輸入**   | 24V DC 直流電輸入                       |
| **網路介面**   | 10/100 Base-T Ethernet (RJ45)           |
| **擴充插槽**   | 4 槽 (Slot C1 ~ C4)                     |
| **插槽介面**   | IDC 50P, 3.3V DIO                       |
| **支援軟體**   | LabVIEW, Python, C#, WPC Device Manager |
| **尺寸**       | (請參閱各模組外觀圖)                    |
