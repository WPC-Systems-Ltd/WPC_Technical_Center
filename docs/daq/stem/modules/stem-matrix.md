# 4x6 Matrix Switch Module

本模組提供 4x6 矩陣開關功能，可自由切換 Row 與 Column 的連接狀態。

##

*   **外部電源**：24V DC
*   **矩陣規模**：Row 4 x Column 6
*   **開關元件**：機械式繼電器 (Mechanical Relays)
