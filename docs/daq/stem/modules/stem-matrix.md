# 4x6 Matrix Switch 模組

本模組提供 4x6 矩陣開關功能，可自由切換 Row 與 Column 的連接狀態。


## 規格

| 項目              | 規格              |
| :---------------- | :--------------- |
| **外部電源**      | 24 VDC            |
| **連接器**        | IDC 50P Connector |
| **矩陣規模**      | Row 4 x Column 6  |
| **開關元件**      | 機械式繼電器       |
