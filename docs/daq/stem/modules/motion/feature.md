# 4軸運動控制模組

本模組提供 4 軸運動控制功能，支援脈波指令 (Pulse Command) 控制。

## 特色

*   **控制模式**：Pulse/Direction 或 CW/CCW
*   **輸出頻率**：差動輸出 (Differential output) 最高 8 MPPS
*   **編碼器輸入**：支援 ABZ 相 (Max 12 MHz)
*   **極限與原點**：支援 Home / FWD / REV limit 輸入
*   **伺服控制**：支援 Servo On / Alarm Clear 輸出
*   **進階功能**：
    *   斷點輸出 (Breakpoint) 與 位置捕捉 (Position Capture)
    *   梯形與 S 曲線速度規劃 (Trapezoidal and S-curve velocity profile)
    *   直線/圓弧/螺旋插補 (Linear/circular/helical interpolation)
    *   多軸同步移動 (Multi-axes sync move)
