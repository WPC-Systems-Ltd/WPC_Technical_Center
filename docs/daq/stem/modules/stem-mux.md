# 24ch 2-wire Multiplexer Module

本模組提供 24 通道多路復用功能，適用於多點訊號切換量測。

## 特色

*   **外部電源**：24V DC
*   **通道數**：24 ch (2-wire)
*   **開關元件**：機械式繼電器 (Mechanical Relays)
