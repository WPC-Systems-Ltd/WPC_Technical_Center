### LED 指示燈與按鈕說明

#### 0️⃣ 重置按鈕
- 直接按下可重新啟動裝置。
- 同時按下 **FUNC** 按鈕與重置按鈕，裝置將重新啟動並進入 Bootloader 模式。

#### 1️⃣ LED1 狀態指示燈
- 若主程式初始化無誤，啟動時會閃爍兩次。
- 系統運作時會恆亮。
- 若發生錯誤，將以特定週期閃爍。

#### 2️⃣ LED2 作業系統指示燈
- 若 Bootloader 初始化無誤，啟動時會閃爍兩次。
- Bootloader 運作時會恆亮。
- 作業系統正常運作時會週期性閃爍。
- 連接乙太網路線時，閃爍頻率約為 4 Hz。
- 未連接乙太網路線時，閃爍頻率約為 2 Hz。
- 若作業系統停止運作，此燈將熄滅。

#### 3️⃣ FUNC 按鈕
- 長按約 3 秒可將 IP 重置為預設值 `192.168.1.110`。
- 短按可切換 OLED 面板顯示頁面。

#### 4️⃣ OLED 面板
- 顯示網路資訊、韌體版本、序號及其他功能資訊。

#### 5️⃣ 24V DC 電源輸入
#### 6️⃣ 保護接地 (PE)
#### 7️⃣ 10/100M 乙太網路孔

## 產品選型指南

| 分類         | 型號           | 主要功能特色                                       |
| :----------- | :------------- | :------------------------------------------------- |
| **類比輸入** | **Ethan-A**    | 8 通道 16-bit 電壓輸入 (±10V)                      |
|              | **Ethan-A2**   | 8 通道 24-bit 電壓輸入 (±13.75V)                   |
|              | **Ethan-I**    | 8 通道 24-bit 電流輸入 (±20mA)                     |
|              | **Ethan-IA**   | 8 通道 24-bit 電壓/電流輸入                        |
|              | **Ethan-T**    | 7 通道熱電偶輸入 (支援 K, J, N, R, S, T, E, B 型)  |
| **類比輸出** | **Ethan-O**    | 8 通道 16-bit 電壓輸出 (±10V), 支援波形產生        |
| **數位 I/O** | **Ethan-D**    | 6 通道 Relay (NPN 驅動), 8 通道 DI, 2 通道 PWM     |
|              | **Ethan-EX-D** | 16 通道 DO, 16 通道 DI, 2 通道 PWM, 2 通道 Counter |
|              | **Ethan-L**    | 6 通道機械式繼電器 (4x Form-A, 2x Form-C)          |
| **其他**     | **Ethan-P**    | 4 通道可變數位電位器 (300Ω-50kΩ)                   |

> 💡 **注意**：部分型號詳細規格請參閱 [產品規格書](./specification.md)。
