1. **Reset Button**

- Pressing it directly will restart the device.
- Pressing the FUNC button simultaneously will restart the device and enter bootloader mode.

2. **LED1 (Status Indicator)**

- If there are no errors during the main program initialization, it will blink twice at startup.
- It remains illuminated when the system is running.
- It blinks in a specific periodic pattern if an error occurs.

3. **LED2 (Operating System Indicator)**

- If there are no errors during bootloader initialization, it will blink twice at startup.
- It remains illuminated when the bootloader is running.
- It blinks periodically when the operating system is functioning normally.
- If an Ethernet cable is connected, it blinks at approximately 4 Hz.
- If the Ethernet cable is not connected, it blinks at approximately 2 Hz.
- It does not blink if the operating system has stopped running.

4. **FUNC Button**

- Pressing and holding the FUNC button for about 3 seconds will reset the IP to the default value of 192.168.1.110.
- A short press switches to the next display page on the OLED panel.

5. **OLED Panel**

- Displays network information, firmware information, serial number, and other functional information.

6. **24V DC Input**

7. **Protective Ground (PE)**

8. **10/100M Ethernet**
