# cPCI 8ch-RS232

## 1. 簡介

---

cPCI 8ch-RS232 是一張具備 8 通道 RS232 的通訊介面卡。

## 2. 特色

---

*   通道數：8 通道
*   連接器：D-SUB 37P
*   隔離：每個通道之間彼此相互隔離。
*   Baud Rate：最高 Baud rate 可達到 500kbps 左右。
