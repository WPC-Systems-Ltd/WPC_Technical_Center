# cPCI 8ch-RS485 422

## 1. 簡介

---

cPCI 8ch-RS485 422 是一張具備 8 通道 RS485 或 RS422 通訊卡。

## 2. 特色

---

*   通道數：8 通道
*   連接器：D-SUB 37P
*   隔離：通道與通道之間彼此隔離。
*   模式：RS-485 與 RS-422 模式可透過程式切換，每個通道可設定不同模式。
*   Baud Rate：最高 Baud rate 可達到 500kbps 左右。
