# cPCI 32ch-DO

## 1. 簡介

---

cPCI 32ch-DO 是一張具備 32 通道輸出的 24V 工業隔離數位 I/O 卡。

## 2. 特色

---

*   通道數：32 通道輸出
*   隔離：24V 工業隔離
*   連接器：D-SUB 37P
*   最大輸出電流：每個 PIN 腳最大輸出電流為 350mA。
