# cPCI 2ch-CAN FD

## 1. 簡介

---

cPCI 2ch-CAN FD 是一張具備 2 通道的 CAN FD (Controller Area Network Flexible Data-Rate) 卡。

## 2. 特色

---

*   通道數：2 通道
*   連接器：D-SUB 37P
*   隔離：兩個通道彼此之間是隔離的。
*   終端電阻：內建 120 歐姆終端電阻。
