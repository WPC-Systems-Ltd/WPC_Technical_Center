# cPCI 8ch-LPCS

## 1. 簡介

---

cPCI 8ch-LPCS 是一張具備 8 通道的可程式電流源模組卡片。

## 2. 特色

---

*   通道數：8 通道
*   最大輸出：0.45 安培 (0.45A)
*   電源需求：需從 Vcs 腳位外部供電，最高電壓可達 48VDC。
*   監控功能：可同時監控各通道的輸出電壓和電流，使使用者能夠自行進行補償，以確保輸出的穩定性。
