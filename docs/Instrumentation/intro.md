# gStack-RIO 產品介紹

## 1. 簡介

---

WPC 自行研發的 gStack 產品系列採用 Rack 機箱設計，專為分散式、高密度 I/O 應用而設計。每個機箱擁有 8 個插槽，提供使用者靈活搭配各種模組的選項。

前置面板設計的模組卡片可以快速抽換，這對於系統維修、更換以及未來功能擴充極為便利。使用 gStack-RIO 驅動的用戶可以通過 Host PC 及 Real-time 應用程式進行 LabVIEW 開發，實現高效控制與管理。

## 2. gStack-RIO 機箱

---

gStack-RIO 機箱配置如下：
*   左側：設有一個 gStack-RIO 控制器。
*   右側：搭載 8 個可根據需要自由搭配的 gStack-RIO 模組卡片。
*   電源：24VDC 電源通過左下方的歐規端子台（綠色）輸入。大部分模組卡片無需額外供電，僅少數電源類型的卡片需要外部供電支援。

控制方法可選擇使用 Ethernet 遠端遙控（PC Remote），或者透過 LabVIEW Real-time 開發獨立運行的應用程式（Stand-alone）。這種設計不僅提供了靈活性，還能有效支持各種分散式、高密度 I/O 應用需求。
